# 📧 Spam Email Detection (ASP.NET + ML.NET + SQL Server)

## 🚀 Overview
This project is a simple **Spam Email Detection System** that classifies emails as:
- Spam (1)
- Not Spam (0)

Built using **ASP.NET Core, ML.NET, and SQL Server**.

---

## 🛠️ Tech Stack
- ASP.NET Core Web API  
- ML.NET  
- SQL Server  
- Entity Framework Core  

---

## 🧠 How It Works
- Input: Email message  
- Model: Binary Classification (Logistic Regression)  
- Output: Spam / Not Spam with confidence score
